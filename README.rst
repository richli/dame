====
Dame
====

Dame is a `SIR <http://www.mers.byu.edu/SIR.html>`_ file viewer. It's focused on comparing and interpreting two SIR files together.

Dame is distributed under the MIT license.

Dependencies
============

* Python 2
* Numpy
* PyQt4
* libsir (not yet)

Features
========

Dame can:

* Load a SIR image
